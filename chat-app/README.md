# Chat App Full Stack Template

A starter realtime chat app with:
- **Backend**: Node.js, Express, Socket.io, MongoDB, JWT auth
- **Frontend**: React (Vite), Zustand, Socket.io client

## Run locally

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Default URLs:
- API: http://localhost:5000
- Web: http://localhost:3000
