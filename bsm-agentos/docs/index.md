# BSM-AgentOS Documentation

BSM-AgentOS is a personalized agent ecosystem scaffold focused on:

- safe agent execution orchestration,
- centralized security policy enforcement,
- observability through a lightweight dashboard,
- predictive reliability scoring.

## Run locally

```bash
cd bsm-agentos
pip install -r requirements.txt
pytest -q
streamlit run bsm_agentos/dashboard/main.py
```
